# JeroScaffold

#### 介绍
JeroScaffold是基于kotlin+viewBinding+viewModel的一套Android快速开发模板

#### 软件架构

- scaffold-core：快速开发的基础脚手架，由ScaffoldActivity、ScaffoldFragment、ScaffoldViewModel组成
- scaffold-ext：一些常用的扩展库
- scaffold-ui：一些ui相关的的扩展库
